About

I’m a Senior Software Developer with a Mechanical Engineering background and 16+ years of experience building custom plugins and automation tools for AutoCAD, SolidWorks, and Revit.

Most CAD developers are software engineers who learned CAD. I’m a mechanical engineer who became a software engineer — so I read your engineering problem first and design the right tool for it, instead of building a clever plugin nobody uses.

What I do:

🔧 Custom Plugin & Add-in Development AutoCAD (ObjectARX C++, .NET, VBA, AutoLISP), SolidWorks API, Revit API. From a 2-day macro to a multi-month commercial product — fixed scope, fixed price, source code on delivery.

☁️ Autodesk Cloud Integration APS (Forge): Data Management, Model Derivative, Design Automation, Webhooks. ACC: Issues, Docs, Model Coordination, Tasks. Connect your CAD desktops to ERP, PLM, and BI dashboards.

⚙️ CAD Process Automation Batch exports (DWG → PDF, RVT → IFC/NWC, SLDDRW → PDF), bulk parameter updates, BOM generation, drawing standard enforcement, file management. Replace hours of manual work with one click.
